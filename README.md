## About
It is a ls-command analogue made at EPITECH's course Unix system programming.


It reproduces ls behavior with flags:

 - t – time sorting
 - r – reverse order sorting
 - a – show hidden files
 - R – display the list of subfolders
 - l – show more details

## Compilation
Compilation via Makefile

    make

 ## Usage


    ./my_ls -lR /bin /dev
